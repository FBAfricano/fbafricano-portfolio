About this project

A lightweight Python compiler that demonstrates the core stages of compilation — lexing, parsing, AST construction, and evaluation. Designed as a beginner-friendly project for understanding how compilers and interpreters process code.

Features & Functions

  • ◆ Lexical Analysis: Processes identifiers, literals, and operators to prepare code for parsing.
  • ◆ Syntax Parsing: Converts code into structured AST (Abstract Syntax Tree) nodes for evaluation.
  • ◆ AST Evaluation & Variable Storage: Evaluates AST nodes and manages variable values during execution.
  • ◆ Modular Architecture: Separates each compiler phase into distinct modules for clarity and maintainability.
  • ◆ Error Handling: Provides clear and informative messages for syntax or logical issues in the code.
