About this project

Taskendar is a personal productivity app I am currently developing. It combines a calendar-based layout with an intuitive task management system, allowing users to plan, track, and visualize their daily activities more naturally. The project is built from scratch using React, and I am gradually designing each interface and feature to create a smooth and organized workflow for everyday planning.

Features & Functions

  • ◆ Homepage / Main View: Displays the current month’s calendar, with each day showing tasks scheduled for that date. Clicking a day opens a panel with all tasks, completion checkboxes, and editing options.
  • ◆ Calendar Features: Shows the current month by default and can expand to a multi-month view similar to the iOS Calendar, allowing users to quickly switch between months.
  • ◆ Task System: Supports tasks with start dates and optional deadlines. Tasks appear on their start date, while deadline days display “Deadline for: [Task Name]”. Tasks can be completed, edited, or moved to new dates.
  • ◆ Progress Dish: A circular progress indicator displayed persistently on the homepage, showing daily task completion and listing all tasks for the current day beneath it.
